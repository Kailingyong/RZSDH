################ Contents ################
This package contains cleaned up codes for the RZSDH, including:

RZSDH_demo.m: test example on public MIRFlickr dataset
RZSDH.m: function to optimize the objective function of RSMFH
Noiselabel_function_3.m: function to noise label processing
Kernelize.m: function to compute a kernel matrix
map_rank1:function to calculate mAP value
